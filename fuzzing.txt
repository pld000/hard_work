1. Фаззинг система
Мною был выбран Jazzer.js. Фаззинг тестирование для node js. Много лет пишу код на javascript, поэтому jazzer.js
был интуитивно понятен.

2. Проект
Для фаззинга использовался сервер на node js, одного из проектов, на котором я работаю. Помимо обычного server side
rendering, сервер выполняет функции по генерации темы магазина (пользователь вводит в админке два главных цвета, а на их
основе формируются множество других цветов, тона, полутона, различные css эффекты, и после, выстраивается цветовая тема
магазина). Это реализуется модулем с утилитарными функциями на 600 строк кода.

3. Тестирование
Использовался фаззинг бинарных данных. 80% функций падали из за невалидности входных данных, что я исправил с помощью
блоков try catch и понятным логированием ошибки

4. Трудности
Тема показалось немного трудноватой в реализации, пока я не нашел jazzer.js с которым смог справиться.
